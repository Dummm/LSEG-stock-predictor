# LSEG Stock Predictor

## Running steps
```bash
# Setup venv
python3 -m venv .venv

# Enable venv
# Linux
source .venv/bin/activate
# Windows
.venv/Scripts/activate

# Install pandas
pip install pandas

# Run script
# Check --help for arguments
python3 main.py 

```
